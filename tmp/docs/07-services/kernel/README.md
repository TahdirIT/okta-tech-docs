# Kernel Services

Kernel services provide foundational capabilities that are used across multiple domains. These services are typically infrastructure-focused and support the core operations of the system.

## Services

- [Service Template](service-template/description.md) - Template for creating new service documentation

## Characteristics

- **Reusable**: Used by multiple domain services
- **Infrastructure-focused**: Provide foundational capabilities
- **Cross-cutting**: Support operations across different business domains
- **Stable**: Core services with well-defined interfaces
