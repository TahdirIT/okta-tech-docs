# تجربة المستخدم - تسجيل الكيان

هذا المجلد يوثق تجربة المستخدم (UX) لتدفق **تسجيل كيان جديد**.

## الصفحات

تسجيل كيان جديد يجب أن يكون عبر صفحتين فقط:

- **Landing Page**: صفحة تعريفية تحتوي على خيار/زر واضح مثل “تسجيل كيان جديد”.
- **Login Page**: تحتوي على تدفق التسجيل (Wizard) لإنشاء الكيان + حساب المالك، ثم الانتقال إلى تجربة التحقق (إن كانت مطلوبة حسب الدولة).
