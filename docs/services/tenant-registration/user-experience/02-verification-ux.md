# تجربة المستخدم - التحقق ومنع الاستخدام قبل التفعيل

هذا المستند يصف تجربة المستخدم بعد إنشاء الحساب/الكيان وقبل إتمام التحقق.

المرجع:

- `docs/services/tenant-registration/guide/04-verification-gating.md`

## شاشة “الحساب غير مفعّل”

### محتوى الشاشة

- رسالة واضحة: “يلزم التحقق من الجوال أو البريد قبل استخدام النظام”.
- عرض متطلبات التحقق حسب الدولة:
  - قد يكون الجوال فقط / البريد فقط / كلاهما / أي واحد.

### أزرار/إجراءات

- بدء/إعادة إرسال رمز الجوال (OTP) (إن كانت القناة متاحة).
- بدء/إعادة إرسال رابط/رمز البريد (إن كانت القناة متاحة).
- تغيير الجوال/البريد (إن كانت السياسة تسمح).
- تسجيل الخروج.

## التعامل مع حالات شائعة

- **انتهت صلاحية الرمز**: رسالة + زر إعادة إرسال.
- **تم تجاوز عدد المحاولات**: رسالة + مدة انتظار (Rate limiting).
- **قناة غير متاحة**: إن كان الجوال غير مطلوب ولم يدخل المستخدم رقمًا، تعرض الواجهة خيار البريد فقط (أو تطلب إدخال رقم جوال قبل بدء التحقق).

## بعد نجاح التحقق

- إظهار رسالة نجاح.
- تحويل المستخدم إلى الصفحة الرئيسية/لوحة التحكم.
- (اختياري) عرض “خطوات بدء سريعة” للـ Tenant الجديد (Onboarding).
