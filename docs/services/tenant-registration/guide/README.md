# تسجيل الكيان (Tenant Registration)

هذا المجلد يحتوي على توثيق شامل لخدمة **تسجيل الكيان** (إنشاء Tenant جديد) في المنصة.  
التوثيق يغطي:

- تدفق التسجيل للمستخدم النهائي (Wizard).
- الحقول المخصصة (Custom fields) التي تُحدد على مستوى الدولة (Country) من قبل المسؤول.
- قواعد إلزامية البريد/الجوال حسب إعدادات الدولة.
- مرحلة التحقق (Mobile/Email verification) كشرط قبل تفعيل الحساب/الكيان.
- **أحداث (Events)** تُطلق أثناء التسجيل لتسهيل التكامل بنمط Event-Driven مع خدمات مستقبلية.
