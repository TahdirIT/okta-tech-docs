# الهدف من الميزة

## الأهداف

- توفير واجهة واحدة لطلب OTP والتحقق منه واستهلاكه.
- تقليل الاحتيال/الإساءة عبر حدود طلبات ومعايير أمان موحدة.
- دعم حالات الاستخدام الشائعة:
  - تسجيل الدخول/التحقق بخطوتين (2FA/Step-up).
  - إنشاء حساب/تأكيد البريد أو الجوال.
  - استعادة كلمة المرور.

## خارج النطاق (حالياً)

- إدارة قوالب الرسائل بالتفصيل (يُفضل أن تكون ضمن نظام الإشعارات/القوالب).
- دعم مفاتيح/أجهزة FIDO أو Passkeys (هذه آليات مختلفة عن OTP).
