# إدارة المناطق

ضمن إعدادات الدولة يمكن إدارة التقسيمات الجغرافية للدولة، والتي يتم تحميلها/عرضها على شكل هيكل هرمي:

- **المناطق** (Regions)
  - **المدن** (Cities)
    - **الأحياء** (Districts)

> **ملاحظة حول حزمة `nnjeim/world`:** البيانات الأساسية للمناطق والمدن تأتي من حزمة `nnjeim/world`. في البكج يُطلق على المناطق اسم "states" (الولايات/المحافظات)، بينما في المشروع نستخدم مصطلح "regions" (المناطق) للتوافق مع المصطلحات المحلية. الأحياء (districts) هي مستوى إضافي خاص بالمشروع ولا يوفرها البكج.

## ما الذي توفره الإدارة?

- **استيراد المناطق والمدن** من حزمة `nnjeim/world` للدولة المحددة
- **استعراض المناطق** التابعة للدولة
- **بحث** داخل المناطق والمدن
- **إضافة/تعديل/حذف** (بحسب الصلاحيات المتاحة)

## بيانات شائعة

### بيانات المنطقة (من البكج)

- **اسم المنطقة** (`name`) — من البكج
- **كود المنطقة** (`state_code`) — من البكج
- **نوع المنطقة** (`type`) — مثل: state, province, region
- **الإحداثيات** (`latitude`, `longitude`) — من البكج

### بيانات إضافية (خاصة بالمشروع)

- **اسم المنطقة بالعربية** (`name_ar`) — إضافة محلية
- **اسم المنطقة بالإنجليزية** (`name_en`) — يمكن استخدامه كـ override
- **معرّف/رقم وطني** (`national_id`) — خاص بالمشروع
- **عدد المدن** التابعة
- **عدد الأحياء** التابعة (قد يُعرض أو يُحسب حسب تصميم الصفحة)
- **الحدود الجغرافية** (`boundaries`) — خاص بالمشروع (يتطلب PostGIS)

## استيراد البيانات من البكج

يمكن استيراد المناطق والمدن للدولة المحددة من حزمة `nnjeim/world`:

```php
use Nnjeim\World\World;

// استيراد المناطق (states) لدولة معينة
$action = World::states(['filters' => ['country_id' => $countryId]]);

// استيراد المدن (cities) لدولة أو منطقة معينة
$action = World::cities(['filters' => ['country_id' => $countryId, 'state_id' => $stateId]]);
```

بعد الاستيراد، يتم إضافة الحقول الإضافية المطلوبة للمشروع (مثل `name_ar`, `national_id`, `ulid`).

## استخدامات إدارة المناطق

- **ربط المدارس والجهات** بمواقعها الجغرافية
- **التقارير والإحصاءات** حسب المنطقة/المدينة
- **توحيد البيانات المرجعية** داخل الدولة
- **استيراد البيانات الجغرافية** من مصادر موثوقة (البكج) لضمان الدقة والاكتمال
