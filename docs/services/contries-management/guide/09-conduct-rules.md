# قواعد السلوك والمواظبة

تدعم إعدادات الدولة إدارة **قواعد السلوك والمواظبة** بهدف توحيد آلية التعامل مع:

- **الغياب** (قواعد الغياب والإجراءات)
- **المشكلات السلوكية** (درجات/تصنيفات المشكلات والإجراءات التربوية)

## السجل (Conduct Rules Register)

يتم إنشاء سجل/سجلات لقواعد السلوك على مستوى الدولة، ويتم عرضها ضمن إعدادات الدولة. كل سجل يمثل “حزمة قواعد” يمكن الرجوع لها وإدارتها.

## مجموعات الدرجات (Conduct Problem Grade Groups)

داخل السجل يتم إنشاء مجموعات لتجميع الدرجات/القواعد، وقد ترتبط بالمراحل/الصفوف بحسب تصميم الدولة.

## درجات المشكلات (Conduct Problem Grades)

لكل مجموعة يمكن تعريف درجات (Levels/Grades) للمشكلات السلوكية، ثم ربط:

- **مشكلات سلوكية (Conduct Problems)**
- **إجراءات تربوية (Conduct Educational Procedures)**

## قواعد الغياب (Absence Rules) وإجراءات الغياب (Absence Procedures)

يمكن تعريف قواعد مرتبطة بالغياب (مثل حد معين من الأيام) ثم إجراءات يتم تنفيذها عند تحقق الشرط (تحذير، إشعار، إجراء إداري…).

## استخدامات قواعد السلوك والمواظبة

- **توحيد تطبيق السياسات** داخل الدولة
- **رفع الانضباط** وتقليل الاجتهادات الفردية
- **تسهيل التقارير** والتحليلات المتعلقة بالمواظبة والسلوك
