# التقويم

تدعم المنصة تعريف التقويم الدراسي لكل دولة بما يتوافق مع الأنظمة الرسمية الصادرة عن الوزارة التعليمية أو الجهة المنظمة.

## عناصر التقويم الدراسي

يجب أن يشمل التقويم على مستوى الدولة ما يلي:

- **الأعوام الدراسية** (بداية/نهاية) والفصول التابعة لها
- **بداية ونهاية كل فصل** (Term)
- **جدول الإجازات والأنشطة** (Holiday Schedule)
- **أيام الدراسة الفعلية** وأيام نهاية الأسبوع (مرتبطة بإعدادات الأسبوع)

## استخدامات التقويم

يتم استخدام التقويم الدراسي المرجعي في:

- **إنشاء الخطط الدراسية**
- **ضبط الجداول الزمنية**
- **حساب أيام الدوام والإجازات**
- **تنظيم الاختبارات والأحداث الأكاديمية**
- **توحيد الإعدادات بين الجهات التعليمية داخل الدولة**

## جدول الإجازات والأنشطة (Holiday Schedule)

ضمن تبويب **التقويم** في إعدادات الدولة يمكن إنشاء/تعديل عناصر الجدول، ويشمل كل عنصر:

- **العنوان** (يدعم العربية/الإنجليزية)
- **تاريخ البداية** و**تاريخ النهاية**
- **النوع**:
  - `HOLIDAY`: إجازة
  - `ACTIVITY`: نشاط/حدث
- **تسجيل الغياب خلال الفترة** (`register_absent`)
- **إرسال إشعارات** (`send_notifications`) مع:
  - **موعد الإرسال** (`send_notifications_at`)
  - **محتوى الإشعار** (`notification_content`)
  - **المستفيدون** (`notified_users`)

### قواعد التحقق

- يجب أن يكون **تاريخ النهاية بعد أو يساوي تاريخ البداية**.
- يتم التحقق (خصوصاً للإجازات) من **عدم تداخل الفترات** ضمن نفس النطاق.
