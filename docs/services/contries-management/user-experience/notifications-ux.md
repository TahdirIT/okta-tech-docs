# تجربة المستخدم - إدارة تعريفات الإشعارات

## الوصول إلى الصفحة

يتم الوصول إلى صفحة إدارة تعريفات الإشعارات من القائمة الجانبية لإدارة الدول، بعد اختيار الدولة المطلوبة.

## عرض تعريفات الإشعارات

تظهر الصفحة قائمة بجميع تعريفات الإشعارات مع إمكانية التصفية حسب:
- المجموعة (Group)
- نوع الإشعار
- حالة التفعيل

كل تعريف يعرض: المعرف (Slug)، العنوان، المجموعة، وحالة قنوات الإرسال (app/whatsapp/sms).

## إنشاء تعريف إشعار جديد

1. المستخدم ينقر على "إضافة تعريف إشعار جديد"
2. يفتح نموذج يحتوي على: المجموعة (من قائمة منسدلة)، المعرف (Slug)، العنوان، المحتوى
3. قسم للمتغيرات (Variables): يمكن تحديد المتغيرات المستخدمة في المحتوى
4. قسم لتكوين قنوات الإرسال: تفعيل/إيقاف كل قناة (app/whatsapp/sms) مع تحديد الحالة (forced/active/inactive)
5. بعد الحفظ، يتم إضافة التعريف إلى القائمة

## إدارة مجموعات الإشعارات

1. من نفس الصفحة، يوجد قسم منفصل أو تبويب خاص بـ "المجموعات"
2. المستخدم يمكنه إنشاء مجموعة جديدة أو تعديل مجموعة موجودة
3. كل مجموعة تحتوي على اسم ووصف
4. يمكن عرض جميع التعريفات التابعة لكل مجموعة

## تكوين قنوات الإرسال

1. من صفحة التعريف، يوجد قسم "قنوات الإرسال"
2. لكل قناة (app/whatsapp/sms)، يمكن:
   - تفعيل/إيقاف القناة
   - تعيين الحالة: forced (إجباري)، active (نشط)، inactive (غير نشط)
3. التغييرات يتم حفظها تلقائياً

## تعديل أو حذف تعريف إشعار

1. المستخدم ينقر على "تعديل" بجانب التعريف
2. يفتح نفس النموذج مع البيانات الحالية
3. يمكن تعديل جميع البيانات بما فيها المحتوى والمتغيرات
4. للحذف، ينقر على "حذف" مع ظهور تأكيد

## معاينة الإشعار

يوجد زر "معاينة" بجانب كل تعريف يعرض كيف سيظهر الإشعار للمستخدمين النهائيين مع إمكانية اختبار المتغيرات.
