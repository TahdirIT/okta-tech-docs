# تجربة المستخدم - التقويم

## الوصول إلى الصفحة

يتم الوصول إلى صفحة التقويم من القائمة الجانبية لإدارة الدول، بعد اختيار الدولة المطلوبة.

## عرض التقويم الدراسي

تظهر الصفحة تقويماً شهرياً أو أسبوعياً يعرض:
- الأعوام الدراسية والفصول التابعة لها
- الإجازات والأنشطة المحددة
- أيام الدراسة وأيام نهاية الأسبوع (مميزة بألوان مختلفة)

يمكن للمستخدم التنقل بين الأشهر أو الأعوام الدراسية باستخدام الأسهم أو القوائم المنسدلة.

## إدارة جدول الإجازات والأنشطة

### عرض الإجازات والأنشطة

يوجد قسم منفصل أو تبويب يعرض قائمة بجميع الإجازات والأنشطة مع:
- العنوان
- تاريخ البداية والنهاية
- النوع (إجازة/نشاط)
- حالة تسجيل الغياب
- حالة الإشعارات

### إضافة إجازة/نشاط جديد

1. المستخدم ينقر على "إضافة إجازة/نشاط"
2. يفتح نموذج يحتوي على: العنوان، تاريخ البداية والنهاية، النوع
3. خيارات إضافية: تفعيل تسجيل الغياب، تفعيل الإشعارات مع تحديد موعد الإرسال والمحتوى
4. بعد الحفظ، يظهر العنصر في التقويم والقائمة

### تعديل أو حذف إجازة/نشاط

1. المستخدم ينقر على العنصر في التقويم أو القائمة
2. يفتح نفس النموذج مع البيانات الحالية
3. بعد التعديل، يتم حفظ التغييرات
4. للحذف، ينقر على "حذف" مع ظهور تأكيد

## عرض الأعوام الدراسية

يوجد قسم منفصل يعرض قائمة الأعوام الدراسية مع إمكانية الانتقال إلى صفحة إدارة الأعوام الدراسية (راجع ملف academic-years-ux.md).
