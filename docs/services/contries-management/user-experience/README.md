يتم الوصول إلى ميزة إدارة الدول من القائمة الرئيسية للمنصة، وتظهر كقسم منفصل يحتوي على جميع الأقسام الفرعية المذكورة أعلاه. كل قسم له صفحة مستقلة مع إمكانية التنقل السريع بين الأقسام عبر قائمة جانبية أو تبويبات.
