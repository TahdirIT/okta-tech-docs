# تجربة المستخدم - قواعد السلوك والمواظبة

## الوصول إلى الصفحة

يتم الوصول إلى صفحة قواعد السلوك والمواظبة من القائمة الجانبية لإدارة الدول، بعد اختيار الدولة المطلوبة.

## عرض قواعد السلوك

تظهر الصفحة قائمة بجميع سجلات قواعد السلوك والمواظبة مع إمكانية التصفية حسب:
- نوع القاعدة
- المجموعة المرتبطة
- حالة التفعيل

## إدارة مجموعات درجات المشكلات

1. يوجد قسم منفصل أو تبويب خاص بـ "مجموعات الدرجات"
2. المستخدم يمكنه إنشاء مجموعة درجات جديدة (مثل: خفيفة، متوسطة، شديدة)
3. كل مجموعة تحتوي على اسم ووصف
4. يمكن ربط درجات المشكلات بالمجموعة

## إدارة درجات المشكلات

1. من قسم "درجات المشكلات"، يمكن إنشاء درجة جديدة
2. النموذج يحتوي على: الاسم، الوصف، القيمة/النقاط، والمجموعة التابعة
3. يمكن تعديل أو حذف الدرجات الموجودة

## إدارة المشكلات السلوكية

1. من قسم "المشكلات السلوكية"، يمكن إنشاء مشكلة جديدة
2. النموذج يحتوي على: الاسم، الوصف، وربط المشكلة بدرجة معينة
3. يمكن ربط عدة مشاكل بدرجة واحدة
4. يمكن تعديل أو حذف المشاكل الموجودة

## إدارة الإجراءات التربوية

1. من قسم "الإجراءات التربوية"، يمكن إنشاء إجراء جديد
2. النموذج يحتوي على: الاسم، الوصف، وربط الإجراء بدرجة معينة
3. يمكن ربط عدة إجراءات بدرجة واحدة
4. يمكن تعديل أو حذف الإجراءات الموجودة

## إدارة قواعد الغياب

1. من قسم "قواعد الغياب"، يمكن إنشاء قاعدة غياب جديدة
2. النموذج يحتوي على: الاسم، الوصف، الشروط (مثل عدد الأيام)، والإجراءات المرتبطة
3. يمكن ربط عدة إجراءات بقاعدة غياب واحدة
4. يمكن تعديل أو حذف القواعد الموجودة

## إدارة إجراءات الغياب

1. من قسم "إجراءات الغياب"، يمكن إنشاء إجراء جديد
2. النموذج يحتوي على: الاسم، الوصف، وربط الإجراء بقاعدة غياب معينة
3. يمكن تعديل أو حذف الإجراءات الموجودة

## عرض العلاقات

يوجد قسم "عرض العلاقات" يعرض بشكل مرئي كيف ترتبط المشكلات والإجراءات بالدرجات، وقواعد الغياب بإجراءاتها، مما يساعد المستخدم على فهم الهيكل الكامل.
