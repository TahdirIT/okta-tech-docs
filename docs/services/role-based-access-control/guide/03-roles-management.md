# إدارة الأدوار

إدارة الأدوار تتيح إنشاء وإدارة الأدوار المرجعية بشكل منفصل عن الصلاحيات. يمكن ربط عدة صلاحيات بدور واحد، مما يوفر مرونة في تنظيم الوصول.

## الميزات الرئيسية

### 1. إنشاء دور جديد

يمكن إنشاء دور جديد يدوياً من خلال:
- **اسم الدور** (`name`): مثل `platform-admin`, `school-manager`
- **النطاق** (`scope`): `tenant` أو `system`
- **الحارس** (`guard_name`): افتراضياً `web`
- **الصلاحيات**: اختيار الصلاحيات المراد ربطها بالدور

### 2. ربط الصلاحيات بالأدوار

عند إنشاء دور جديد أو تعديل دور موجود، يمكن:
- اختيار عدة صلاحيات من قائمة الصلاحيات المتاحة
- عرض نطاق كل صلاحية (tenant/system) لسهولة الاختيار
- ربط الصلاحيات بشكل ديناميكي

### 3. مسح الكود لاكتشاف الأدوار المفقودة

يتم مسح ملفات المشروع لاكتشاف الأدوار المستخدمة في الكود ولكن غير موجودة في قاعدة البيانات:

- البحث في ملفات Blade عن `@role`, `@hasrole`
- البحث في ملفات PHP عن `hasRole()`, `hasAnyRole()`, `assignRole()`, `syncRoles()`
- البحث عن أنماط مثل `role:name` في الكود

النتائج تُحفظ في `storage/app/rbac/missing-rbac.json`.

### 4. استيراد الأدوار المفقودة

بعد المسح، يمكن:
- **استيراد جميع الأدوار المفقودة**: إضافة جميع الأدوار المكتشفة إلى قاعدة البيانات
- **استيراد الأدوار المحددة**: اختيار أدوار محددة من القائمة واستيرادها فقط

### 5. عرض وإدارة الأدوار

- عرض جميع الأدوار في جدول منظم
- عرض الصلاحيات المرتبطة بكل دور
- تصنيف الأدوار حسب النطاق (`tenant`/`system`)
- حذف الأدوار غير المستخدمة

## القواعد والقيود

- يجب أن يكون اسم الدور فريداً ضمن نفس `tenant_id` و `guard_name`
- الأدوار على مستوى النظام (`system`) متاحة لجميع المستأجرين
- الأدوار على مستوى المستأجر (`tenant`) خاصة بكل مستأجر
- لا يمكن حذف دور مرتبط بمستخدمين
- يمكن ربط عدة صلاحيات بدور واحد
- يمكن ربط نفس الصلاحية بعدة أدوار

## استخدام الأدوار في الكود

### في Blade Templates

```blade
@role('platform-admin')
    <button>إعدادات النظام</button>
@endrole

@hasrole('school-manager')
    <a href="/schools">المدارس</a>
@endhasrole
```

### في Controllers

```php
public function index()
{
    if (auth()->user()->hasRole('admin')) {
        // ...
    }
}
```

### في Livewire Components

```php
public function mount()
{
    if (!auth()->user()->hasAnyRole(['admin', 'manager'])) {
        abort(403);
    }
}
```

### تعيين أدوار للمستخدمين

```php
$user->assignRole('platform-admin');
$user->syncRoles(['admin', 'manager']);
$user->removeRole('guest');
```
