# تجربة المستخدم - التحكم في الوصول القائم على الأدوار

هذا المجلد يحتوي على توثيق تجربة المستخدم (User Experience) لميزة التحكم في الوصول القائم على الأدوار، ويوضح كيفية التنقل بين الصفحات وأداء العمليات من قبل المستخدم.

## الملفات

- [إدارة الصلاحيات](./01-permissions-management-ux.md)
- [إدارة الأدوار](./02-roles-management-ux.md)
- [اختيار السياق والأدوار](./03-context-selection-ux.md)

## هيكل التنقل العام

يتم الوصول إلى ميزة التحكم في الوصول من القائمة الرئيسية للمنصة تحت قسم "الإعدادات" (Settings)، وتظهر كقسم منفصل يحتوي على:
- إدارة الصلاحيات (`/settings/permissions`)
- إدارة الأدوار (`/settings/roles`)

كما يتم الوصول إلى صفحة اختيار السياق (`/auth/context`) تلقائياً عند الحاجة لاختيار سياق أو تبديل الأدوار.
