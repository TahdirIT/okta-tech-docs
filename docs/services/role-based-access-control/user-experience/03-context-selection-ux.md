# تجربة المستخدم - اختيار السياق والأدوار

## الوصول إلى الصفحة

يتم الوصول إلى صفحة اختيار السياق تلقائياً في الحالات التالية:
- بعد تسجيل الدخول مباشرة (إذا لم يكن هناك سياق محدد)
- عند محاولة الوصول إلى صفحة محمية بدون سياق نشط
- عند النقر على زر "تبديل السياق" أو "اختيار دور" من القائمة الرئيسية

المسار: `/auth/context`

## عرض الصفحة

### التخطيط العام

الصفحة تحتوي على:
- عنوان رئيسي: "اختر السياق والدور" أو "Select Context and Role"
- قسم للأدوار على مستوى النظام (إن كان المستخدم لديه أدوار على مستوى النظام)
- قسم للأدوار على مستوى المستأجر (إن كان المستخدم لديه أدوار في مستأجرين)

## اختيار سياق النظام

### عرض الأدوار المتاحة

إذا كان المستخدم لديه أدوار على مستوى النظام:
- يظهر قسم بعنوان "الأدوار على مستوى النظام" أو "System Roles"
- يتم عرض كل دور كزر (Button) يحتوي على:
  - عنوان الدور (أو الاسم إذا لم يكن العنوان متوفراً)
  - Badge يوضح النطاق: `SYSTEM`
  - أيقونة أو لون مميز

### الخطوات

1. المستخدم يرى قائمة بالأدوار المتاحة على مستوى النظام
2. يمكن اختيار عدة أدوار (Checkboxes أو Multi-select)
3. بعد اختيار الأدوار المطلوبة:
   - ينقر على زر "تأكيد" أو "Continue"
   - يتم حفظ الأدوار المحددة في الجلسة
   - يتم بناء السياق على مستوى النظام
   - يتم إعادة التوجيه إلى لوحة التحكم (`/dashboard`)

### حالة وجود دور واحد فقط

إذا كان المستخدم لديه دور واحد فقط على مستوى النظام:
- يتم اختياره تلقائياً
- يمكن للمستخدم المتابعة مباشرة أو تغيير الاختيار

## اختيار سياق المستأجر

### عرض المستأجرين المتاحين

إذا كان المستخدم لديه أدوار في مستأجرين:
- يظهر قسم بعنوان "الأدوار على مستوى المستأجر" أو "Tenant Roles"
- يتم عرض قائمة بالمستأجرين المتاحين
- لكل مستأجر:
  - اسم المستأجر
  - الأدوار المتاحة في هذا المستأجر

### الخطوات

1. المستخدم يختار المستأجر من القائمة
2. بعد اختيار المستأجر:
   - يتم عرض الأدوار المتاحة في هذا المستأجر
   - كل دور يظهر كزر (Button) يحتوي على:
     - عنوان الدور (أو الاسم إذا لم يكن العنوان متوفراً)
     - Badge يوضح النطاق: `TENANT`
     - اسم المستأجر
     - أيقونة أو لون مميز
3. يمكن اختيار عدة أدوار (Checkboxes أو Multi-select)
4. بعد اختيار الأدوار المطلوبة:
   - ينقر على زر "تأكيد" أو "Continue"
   - يتم حفظ المستأجر والأدوار المحددة في الجلسة
   - يتم بناء السياق على مستوى المستأجر
   - يتم إعادة التوجيه إلى لوحة التحكم (`/dashboard`)

### حالة وجود مستأجر واحد فقط

إذا كان المستخدم مرتبطاً بمستأجر واحد فقط:
- يتم اختياره تلقائياً
- يتم عرض الأدوار المتاحة مباشرة
- يمكن للمستخدم اختيار الأدوار والمتابعة

### حالة وجود دور واحد فقط في المستأجر

إذا كان المستخدم لديه دور واحد فقط في المستأجر المحدد:
- يتم اختياره تلقائياً
- يمكن للمستخدم المتابعة مباشرة أو تغيير الاختيار

## التبديل بين السياقات

### من سياق النظام إلى سياق المستأجر

1. المستخدم ينقر على زر "تبديل السياق" أو "Switch Context" من القائمة الرئيسية
2. يتم إعادة التوجيه إلى صفحة اختيار السياق
3. المستخدم يختار سياق المستأجر
4. يختار المستأجر والأدوار المطلوبة
5. يتم حفظ الاختيارات وبناء السياق الجديد
6. يتم إعادة التوجيه إلى لوحة التحكم

### من سياق المستأجر إلى سياق النظام

1. المستخدم ينقر على زر "تبديل السياق" أو "Switch Context" من القائمة الرئيسية
2. يتم إعادة التوجيه إلى صفحة اختيار السياق
3. المستخدم يختار سياق النظام
4. يختار الأدوار المطلوبة
5. يتم حفظ الاختيارات وبناء السياق الجديد
6. يتم إعادة التوجيه إلى لوحة التحكم

### تغيير المستأجر (داخل سياق المستأجر)

1. المستخدم ينقر على زر "تبديل السياق" أو "Switch Context" من القائمة الرئيسية
2. يتم إعادة التوجيه إلى صفحة اختيار السياق
3. المستخدم يختار مستأجراً مختلفاً
4. يختار الأدوار المطلوبة في المستأجر الجديد
5. يتم حفظ الاختيارات وبناء السياق الجديد
6. يتم إعادة التوجيه إلى لوحة التحكم

## حالات خاصة

### عدم وجود أدوار

إذا لم يكن المستخدم لديه أي أدوار:
- يظهر رسالة: "ليس لديك أدوار متاحة. يرجى الاتصال بالمسؤول."
- لا يمكن المتابعة

### عدم وجود مستأجرين

إذا لم يكن المستخدم مرتبطاً بأي مستأجر:
- لا يظهر قسم الأدوار على مستوى المستأجر
- يمكن فقط اختيار الأدوار على مستوى النظام (إن كانت متاحة)

### عدم وجود أدوار على مستوى النظام

إذا لم يكن المستخدم لديه أدوار على مستوى النظام:
- لا يظهر قسم الأدوار على مستوى النظام
- يمكن فقط اختيار الأدوار على مستوى المستأجر (إن كانت متاحة)

## التحديثات المستقبلية المقترحة

1. **عرض معلومات إضافية**: إضافة معلومات إضافية عن كل دور (مثل عدد الصلاحيات المرتبطة)
2. **البحث والتصفية**: إضافة إمكانية البحث والتصفية للأدوار والمستأجرين
3. **الاختيار السريع**: إضافة زر "اختيار سريع" لاختيار الدور الافتراضي تلقائياً
4. **التذكير بالاختيار**: إضافة خيار لحفظ الاختيار الافتراضي للمستخدم
5. **عرض الصلاحيات**: إضافة إمكانية عرض الصلاحيات المرتبطة بكل دور قبل الاختيار
