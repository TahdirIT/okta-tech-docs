# تجربة المستخدم - إدارة الأدوار

## الوصول إلى الصفحة

يتم الوصول إلى صفحة إدارة الأدوار من قائمة الإعدادات الرئيسية، أو من خلال الرابط المباشر `/settings/roles`.

## عرض قائمة الأدوار

عند فتح الصفحة، يرى المستخدم:

### القسم الأيسر (إنشاء وإدارة الأدوار)

**بطاقة إنشاء دور جديد:**
- حقل إدخال **اسم الدور** (مطلوب، مثل: `platform-admin`)
- قائمة منسدلة **النطاق** (مطلوب: `tenant` أو `system`)
- قسم **مرجع الصلاحيات**:
  - قائمة بجميع الصلاحيات المتاحة
  - لكل صلاحية: مربع اختيار مع اسم الصلاحية ونطاقها (TENANT/SYSTEM)
  - يمكن تحديد عدة صلاحيات
- زر **إضافة دور** لإنشاء الدور الجديد

**بطاقة مسح واستيراد الأدوار:**
- زر **مسح** (أيقونة بحث) لمسح الكود لاكتشاف الأدوار المفقودة
- عرض **تاريخ آخر مسح** (إن وجد)
- قائمة **الأدوار المفقودة من الكود**:
  - مربعات اختيار لكل دور مفقود
  - مربع اختيار **تحديد الكل** لتحديد جميع الأدوار
- أزرار الإجراءات (تظهر فقط عند وجود أدوار مفقودة):
  - **إضافة جميع المفقودة إلى قاعدة البيانات**
  - **إضافة المحددة إلى قاعدة البيانات** (يظهر فقط عند تحديد أدوار)

### القسم الأيمن (عرض الأدوار)

**جدول الأدوار:**
- عمود **الاسم**: اسم الدور
- عمود **النطاق**: نطاق الدور (`TENANT` أو `SYSTEM`) مع شارة ملونة
- عمود **الصلاحيات**: قائمة الصلاحيات المرتبطة بالدور (مفصولة بفواصل) أو "لا يوجد"
- عمود **الإجراءات**: زر **حذف** لكل دور

**حالات الجدول:**
- عند وجود أدوار: عرض جميع الأدوار مرتبة حسب النطاق ثم الاسم
- عند عدم وجود أدوار: رسالة "لا توجد أدوار بعد"

## إنشاء دور جديد

### الخطوات:

1. المستخدم يملأ حقل **اسم الدور** (مثل: `platform-admin`)
2. المستخدم يختار **النطاق** من القائمة المنسدلة:
   - `tenant`: دور خاص بالمستأجر
   - `system`: دور على مستوى النظام
3. المستخدم يحدد **الصلاحيات** المراد ربطها بالدور:
   - يمكن تحديد عدة صلاحيات من القائمة
   - يتم عرض نطاق كل صلاحية (TENANT/SYSTEM) لسهولة الاختيار
4. المستخدم ينقر على زر **إضافة دور**
5. يتم التحقق من البيانات:
   - يجب أن يكون الاسم فريداً ضمن نفس `tenant_id` و `guard_name`
   - يجب أن يكون الاسم مطلوباً ولا يتجاوز 120 حرفاً
   - يجب التحقق من وجود الصلاحيات المحددة
6. بعد النجاح:
   - يتم إنشاء الدور في قاعدة البيانات
   - يتم ربط الصلاحيات المحددة بالدور
   - يتم إعادة تعيين الحقول
   - يظهر إشعار نجاح: "تم حفظ الدور بنجاح"
   - يتم تحديث الجدول تلقائياً لعرض الدور الجديد

## مسح الكود لاكتشاف الأدوار المفقودة

### الخطوات:

1. المستخدم ينقر على زر **مسح** (أيقونة البحث)
2. يتم تنفيذ عملية المسح:
   - مسح جميع ملفات المشروع (Blade, PHP)
   - البحث عن أنماط الأدوار المستخدمة في الكود
   - مقارنة الأدوار المكتشفة مع الأدوار الموجودة في قاعدة البيانات
   - حفظ التقرير في `storage/app/rbac/missing-rbac.json`
3. بعد اكتمال المسح:
   - يتم تحديث قائمة **الأدوار المفقودة**
   - يتم تحديث **تاريخ آخر مسح**
   - يتم إعادة تعيين التحديدات السابقة
   - يظهر إشعار نجاح: "تم مسح الأدوار بنجاح"

## استيراد الأدوار المفقودة

### الطريقة الأولى: استيراد جميع الأدوار

1. المستخدم ينقر على زر **إضافة جميع المفقودة إلى قاعدة البيانات**
2. يتم استيراد جميع الأدوار المفقودة إلى قاعدة البيانات
3. بعد النجاح:
   - يظهر إشعار نجاح: "تم استيراد الأدوار بنجاح"
   - يتم تحديث الجدول تلقائياً
   - يتم إعادة تعيين قائمة الأدوار المفقودة

### الطريقة الثانية: استيراد الأدوار المحددة

1. المستخدم يحدد الأدوار المطلوبة من القائمة (يمكن استخدام **تحديد الكل**)
2. المستخدم ينقر على زر **إضافة المحددة إلى قاعدة البيانات**
3. يتم استيراد الأدوار المحددة فقط
4. بعد النجاح:
   - يظهر إشعار نجاح: "تم استيراد الأدوار بنجاح"
   - يتم تحديث الجدول تلقائياً
   - يتم إعادة تعيين التحديدات

## حذف دور

### الخطوات:

1. المستخدم ينقر على زر **حذف** بجانب الدور المطلوب
2. يتم التحقق من عدم ارتباط الدور بمستخدمين
3. بعد التأكيد:
   - يتم حذف الدور (Soft Delete)
   - يتم إلغاء ربط جميع الصلاحيات المرتبطة بالدور تلقائياً
   - يظهر إشعار نجاح
   - يتم تحديث الجدول تلقائياً

### القيود:

- لا يمكن حذف دور مرتبط بمستخدمين
- عند الحذف، يتم إلغاء ربط جميع الصلاحيات المرتبطة بالدور تلقائياً

## عرض الصلاحيات المرتبطة بالدور

في جدول الأدوار، يتم عرض الصلاحيات المرتبطة بكل دور في عمود **الصلاحيات**:
- إذا كان الدور مرتبط بصلاحيات: يتم عرض أسماء الصلاحيات مفصولة بفواصل
- إذا لم يكن الدور مرتبط بأي صلاحيات: يتم عرض "لا يوجد"

## حالات خاصة

### عدم وجود أدوار

عند عدم وجود أدوار في النظام:
- يظهر في الجدول رسالة: "لا توجد أدوار بعد"
- تظهر قائمة الأدوار المفقودة فارغة مع رسالة: "لم يتم العثور على أدوار مفقودة. قم بتشغيل المسح للتحديث"

### عدم وجود أدوار مفقودة

عند عدم وجود أدوار مفقودة:
- تظهر رسالة: "لم يتم العثور على أدوار مفقودة. قم بتشغيل المسح للتحديث"
- لا تظهر أزرار الاستيراد

### عدم وجود صلاحيات للربط

عند عدم وجود صلاحيات في النظام:
- تظهر رسالة في قسم **مرجع الصلاحيات**: "لا توجد صلاحيات متاحة"
- لا يمكن تحديد أي صلاحيات للربط بالدور
- يمكن إنشاء الدور بدون صلاحيات (يمكن إضافة الصلاحيات لاحقاً)
