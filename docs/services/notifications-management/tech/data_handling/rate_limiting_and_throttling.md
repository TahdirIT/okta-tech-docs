# التعامل مع البيانات: تحديد السرعة (Rate Limiting) والاختناقات

## مستويات القيود
- منصة: حدود كلية لكل قناة/تكامل (SMTP أو taqnyat-sms-laravel أو Firebase Messaging).
- جهة: حدود محلية لحماية السمعة والقنوات.
- مستخدم: حماية من الإغراق العرضي.

## سياسات
- تعريف سياسات قيود قابلة للتهيئة مع مقياس زمني.
- إجراءات عند التجاوز: تأجيل، إسقاط مضبوط، تبديل قناة، تنبيه مراقبة.
