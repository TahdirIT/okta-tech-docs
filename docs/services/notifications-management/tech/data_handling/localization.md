# التعامل مع البيانات: التوطين (Localization)

## مصادر اللغة
- إعدادات المنصة والدولة والجهة.
- تفضيل اللغة للمستخدم إن وجد.
- كشف تلقائي احتياطي (واجهة/ملف شخصي).

## اختيار اللغة
- تطبيق سياسة اللغة الهرمية.
- إن لم تتوفر اللغة المطلوبة، اختيار أقرب بديل محدد.

## متغيرات القوالب
- تعريف صارم لمفاتيح البيانات المطلوبة لكل تعريف.
- التحقق قبل الإرسال وإرجاع أخطاء تشخيصية واضحة.
