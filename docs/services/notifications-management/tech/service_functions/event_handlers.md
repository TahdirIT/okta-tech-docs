# وظائف الخدمة: معالجات الأحداث (Event Handlers)

## مصادر الأحداث
- أحداث نطاقية (Domain Events) صادرة من الخدمات الأخرى.
- أحداث نظامية (مثل تحقق البريد، إعادة تعيين كلمة المرور).

## التدفق
1. استقبال الحدث مع الحمولة.
2. مطابقة الحدث بتعريف الإشعار المناسب.
3. حساب اللغة الفعلية وفق السياسات.
4. اختيار القالب وتجهيزه بالبيانات.
5. إنشاء إشعار وجدولته للإرسال عبر القناة المناسبة.
