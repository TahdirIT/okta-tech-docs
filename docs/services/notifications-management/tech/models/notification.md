# نموذج: الإشعار (Notification)

يمثل نسخة إرسال فعلية مرتبطة بتعريف/مستلم/قناة:
- التعريف: مرجع إلى تعريف الإشعار.
- المستلم: معرف المستخدم/الجهة/القناة المستهدفة.
- اللغة: اللغة الفعلية المختارة بعد تطبيق السياسات.
- القالب المختار: مرجع إلى القالب المستخدم عند الإرسال.
- الحالة: Pending/Sent/Failed/Cancelled/Queued/Retried.
- مفاتيح الترصّد: معرف التماثل (Idempotency Key)، طابع زمني، مرجع التكامل (SMTP أو taqnyat-sms-laravel أو Firebase Messaging).
