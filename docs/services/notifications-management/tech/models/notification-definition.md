# نموذج: تعريف الإشعار (NotificationDefinition)

يمثل تعريفاً مجرّداً لنوع إشعار معيّن:
- المفتاح: معرف فريد (مثل: user.password.reset).
- الوصف والسياق: ماذا يعني ومتى يُستخدم.
- القنوات المسموحة: Email/SMS/In-App.
- متطلبات البيانات: مفاتيح مطلوبة لتعبئة القالب.
- السياسات: السماح بالإلغاء، الأولوية، معدلات الإرسال.
