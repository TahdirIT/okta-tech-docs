# صفحات إعدادات المنصة: الإشعارات

## تعريفات الإشعارات
- قائمة بالتعريفات (بحث/ترشيح/حالة النشر).
- إضافة/تعديل/حذف تعريف، وتحديد القنوات المدعومة.
- ربط التعريف بقوالب افتراضية لكل قناة.

## قوالب الإشعارات (افتراضي المنصة)
- إدارة قوالب متعددة اللغات لكل قناة.
- حقول أساسية: عنوان/نص/مخطط العناصر/متغيرات ديناميكية/مرفقات.
- المعاينة حسب اللغة والقناة.

## سياسة اللغة حسب الدولة
- اختيار اللغة الافتراضية لكل دولة.
- استعراض وراثة اللغة للجهات ضمن الدولة.
