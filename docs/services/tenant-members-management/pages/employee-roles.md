# صفحة أدوار الموظفين (Employee Roles)

## الهدف

إدارة الأدوار tenant-scoped داخل الجهة، بما يشمل:

- إنشاء دور جديد باسم عربي/إنجليزي.
- تعديل الدور وتحديث صلاحياته.
- حذف الدور (مع معالجة المستخدمين المرتبطين به).

## عناصر الصفحة

- **قائمة الأدوار**:
  - اسم الدور
  - عدد الموظفين المرتبطين
  - إجراءات: تعديل / حذف

- **Role modal**:
  - الحقول: `name_ar`, `name_en`
  - اختيار صلاحيات ضمن مجموعات/تبويبات لسهولة التصفح

## سلوك تعيين الصلاحيات

- الصلاحيات تُعرض مجمعة:
  - حسب “tab/group” منطقي (إعدادات/طلاب/موظفين…)
- حفظ التغييرات:
  - يمنح/يسحب الصلاحيات من الدور بشكل تفاضلي

## حذف الدور

- قبل الحذف يجب:
  - فصل الموظفين عن الدور أو تحويلهم إلى دور بديل (حسب سياسة المنتج)
- ثم حذف الدور tenant-scoped.
