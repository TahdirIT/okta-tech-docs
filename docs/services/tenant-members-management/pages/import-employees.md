# صفحة استيراد الموظفين (Import Employees)

## الهدف

إضافة/تحديث موظفي الجهة عبر ملف، مع:

- معاينة البيانات قبل الحفظ.
- تعيين دور افتراضي/أساسي (مثل teacher role) قابل للاختيار.
- ضبط كلمات مرور افتراضية للموظفين المستوردين (حسب سياسة المنصة).

## تجربة المستخدم

- رفع الملف → Parsing → عرض قائمة الموظفين.
- التحقق:
  - الاسم/الهوية
  - صيغة الهاتف (إن كان مطلوباً)
  - منع تكرار الموظف داخل الجهة
- اختيار “الدور الأساسي” (default role) لتعيينه للمستوردين.
- تنفيذ الاستيراد:
  - Upsert للمستخدمين
  - Upsert للموظفين
  - ربط employee↔tenant
  - تعيين الدور tenant-scoped

## نتائج التنفيذ

- نجاح + العودة إلى قائمة الموظفين.
- Jobs لاحقة:
  - توليد/تعيين كلمات مرور افتراضية
