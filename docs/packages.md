# Packages Used in `okta-web/package.json`

## Dependencies

- `@tailwindcss/vite`: Official Tailwind CSS plugin for Vite. It integrates Tailwind v4 into the Vite build pipeline for fast style processing and development feedback.
- `autoprefixer`: PostCSS plugin that automatically adds vendor prefixes to CSS rules. It helps keep styles compatible across different browsers without manual prefix management.
- `axios`: Promise-based HTTP client for browsers and Node.js. It is commonly used to send API requests, handle responses, and simplify request configuration.
- `concurrently`: Utility for running multiple commands at the same time in one terminal session. It is useful for coordinating parallel development tasks or scripts.
- `flatpickr`: Lightweight JavaScript date and time picker. It provides a polished UI for selecting dates or times in forms and interactive inputs.
- `laravel-vite-plugin`: Laravel’s official Vite integration. It connects Vite with Laravel asset handling, Blade helpers, hot reloading, and production builds.
- `vite`: Modern frontend build tool and dev server. It offers fast local development, efficient bundling, and optimized production output for web assets.
- `spatie/laravel-multitenancy`: Unopinionated Laravel multitenancy package. It helps determine the current tenant, supports single or multiple databases, and includes tools for tenant-aware queues, Artisan commands, and environment setup tasks.
- `spatie/laravel-permission`: Laravel package for managing roles and permissions in the database. It integrates with Laravel gates and supports direct permissions, roles, multiple guards, Blade directives, and authorization checks.
- `nwidart/laravel-modules`: Package for organizing large Laravel applications into modules. Each module can contain its own controllers, models, views, routes, migrations, assets, and tests for a more modular architecture.

## Optional Dependencies

- `@rollup/rollup-linux-x64-gnu`: Prebuilt Rollup binary for Linux x64 GNU systems. It speeds up installs by providing a platform-specific native executable when available.
- `@tailwindcss/oxide-linux-x64-gnu`: Native Tailwind Oxide binary for Linux x64 GNU systems. It improves Tailwind performance by using the optimized Rust-powered engine.
- `lightningcss-linux-x64-gnu`: Prebuilt Lightning CSS binary for Linux x64 GNU systems. It enables fast CSS parsing, transformation, and minification in supported environments.

- `tailwindcss`: Utility-first CSS framework used to build the application’s UI. It provides composable classes for layout, spacing, colors, typography, and responsive design.
