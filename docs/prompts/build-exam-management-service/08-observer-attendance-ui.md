العنوان: واجهة المراقب لتسجيل الحضور أثناء الجلسة

المطلوب:
- ضمن @okta-web، أنشئ واجهة للمراقب (المعلم) للوصول إلى جلساته اليومية (حسب اليوم والفترة واللجنة).
- عرض قائمة الطلبة في اللجنة المحددة مع حالة الحضور الحالية.
- تنفيذ إجراءات سريعة:
  - وضع حالة طالب: حاضر/متأخر (<15 دقيقة)/متأخر (>15 دقيقة)/غائب
  - Mark all present (مع تأكيد)
- دعم العمل بزمن حقيقي مبسط (تحديثات مرئية بعد الحفظ)، أو تحديثات مجمعة مع تنبيهات نجاح/فشل.

اعتبارات:
- التقيّد بالأذونات الخاصة بالمراقب.
- تصميم عملي جداً على الجوال والتابلت (حسب النقاط القياسية).
- RTL افتراضياً عند العربية، وإظهار رسائل واضحة.
