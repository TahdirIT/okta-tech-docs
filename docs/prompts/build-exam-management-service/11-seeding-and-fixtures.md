العنوان: بيانات أولية (Seeding) وتركيبات اختبارية (Fixtures)

المطلوب:
- أنشئ Seeders وFactories تغطي:
  - مدرسة/مراحل/شُعب أساسية (إن لم تكن موجودة ضمن بيانات عامة للمشروع)
  - فترة اختبار تجريبية مع أيام وفترات يوم
  - لجان مع مقاعد مهيأة بعدة أحجام
  - مواد اختبارية مرتبطة بفترات اليوم
  - مراقبون مرتبطون بجلسات
  - طلبة موزعون على المقاعد
- وفّر سكريبت واحد لتعبئة بيانات متكاملة تسهّل اختبار الواجهات والتقارير بسرعة.

اعتبارات:
- اجعل الأحجام قابلة للضبط عبر env/params (عدد لجان/مقاعد/طلاب).
- راعِ الأداء عند إنشاء أعداد كبيرة.

المخرجات:
- Factories + Seeders قابلة للتشغيل، موثقة في README الـ Module.
