العنوان: واجهة الطالب لعرض الجدول الاختباري

المطلوب:
- ضمن @okta-web، أنشئ صفحة للطالب لعرض مواده ضمن الفترة النشطة:
  - أيام الاختبار
  - الفترات الزمنية
  - المادة المقررة
  - لجنة الجلوس ورقم المقعد إن توفّر
- دعم التبديل بين العربية والإنجليزية، مع RTL للعربية.
- عرض الحالات الفارغة عندما لا توجد فترة مفعّلة أو لا توجد مواد للطالب.
