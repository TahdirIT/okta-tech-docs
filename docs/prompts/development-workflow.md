# خطة التطوير - إدارة الدول (Countries Management)

## 1. هيكل قاعدة البيانات

### المهام
- إنشاء migrations للجداول الأساسية: `countries`, `regions`, `cities`, `districts`
- إعداد migrations للتقويم: `academic_years`, `terms`, `holiday_schedules`
- إنشاء migrations للهيكل الأكاديمي: `education_level_groups`, `education_levels`, `education_level_group_levels`
- إعداد جدول `subjects` على مستوى الدولة
- إنشاء جدول `notification_definitions`
- إعداد جدول `model_settings` للإعدادات العامة
- إضافة فهارس (indexes) على `country_id` في الجداول التابعة
- استخدام `jsonb` للترجمات والإعدادات في PostgreSQL
- إعداد PostGIS للحدود الجغرافية (اختياري)

### التفاصيل
- الاعتماد على حزمة `nnjeim/world` للبيانات الجغرافية الأساسية
- استخدام ULID كمعرّفات رئيسية
- إضافة soft deletes حيث يلزم
- إضافة timestamps تلقائية
